# Ship Composer API

### Setup

```bash
npm install
```

### Development with nodemon and tsc --watch

```bash
npm start
```


### Run without nodemon and tsc --watch

```bash
npm run serve
```

## Swagger

Visit `http://localhost:3000/swagger` to view the OPENAPI document in Swagger-UI
